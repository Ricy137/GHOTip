# get started

## install
```bash
npm/yarn/pnpm install
```

## setup environment variables
create a `.env` file according to `.env.example`

## common commands

```shell
npx hardhat help
REPORT_GAS=true npx hardhat test
npx hardhat node
npx hardhat clean
npx hardhat compile
npx hardhat run scripts/deploy.js
npx hardhat verify
```
